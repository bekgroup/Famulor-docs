famulor.de bietet mehrere Möglichkeiten, Telefonnummern für eingehende oder ausgehende Anrufe zu handhaben. Sie können eine **dedizierte Nummer** von uns **mieten**, einen **SIP**-Trunk verwenden (keine monatliche Gebühr, aber Carrier-Kosten sind auf Ihrem eigenen PBX), oder eine **bestehende Caller ID** verifizieren (auch keine monatliche Gebühr, aber regionsbasierte Gebühren pro Minute).

---

## 1. Eine dedizierte Nummer mieten

Wenn Sie eine vollständig verwaltete eingehende/ausgehende Leitung möchten, können Sie sie direkt in der Plattform mieten:

- **Monatliche Gebühr**: Typischerweise €3,99/Monat für Standard-Loknummern (US/CA/UK). Regionen variieren:
  - **Europa**: Oft €2-€8/Monat basierend auf Land.
  - **Lateinamerika, Asien, Afrika**: €9+ in einigen Fällen, abhängig von Verfügbarkeit oder lokalen Steuern
- **Benutzerdefinierte Bereitstellung**: Wenn Sie Ihr gewünschtes Land oder Ihre Region nicht aufgelistet sehen, kann unser Team eine Nummer nach Verifizierung lokaler Dokumente benutzerdefiniert bereitstellen.

**Gebühren pro Minute**

- Ausgehende Anrufe auf einer gemieteten Nummer: hängt vom Land ab, das Sie anrufen—könnte von €0,05/Min bis €0,45/Min reichen, falls es ein teures Ziel ist.
- Eingehende Anrufe: normalerweise keine zusätzlichen Kosten, es sei denn, es ist ein spezielles/Gebühren-Szenario in dieser Region.

---

## 2. SIP-Integration (Keine monatliche Gebühr)

Wenn Sie bereits eine VOIP/PBX-Lösung haben, können Sie über einen **SIP-Trunk** mit **keiner monatlichen Miete** von famulor.de integrieren. Stattdessen zahlen Sie:

- **SIP pro Minute**: €0,0004/Min auf unserer Seite für Carrier-Bridging. Ihr VOIP-Anbieter kann auch separate Nutzungsgebühren haben.
- **Ihre bestehende Nummer**: Sie verwenden weiterhin Ihre eigenen Telefonleitungen für eingehende und ausgehende Anrufe, aber die Carrier-Ebene kostet €0,0004/Min bei uns, wann immer die KI am Anruf ist.

Dies ist ideal, wenn Sie sich mit der Verwaltung Ihres eigenen Telefonsystems wohlfühlen oder Ihre bestehenden Nummern ohne Miete von uns behalten möchten.

---

## 3. Caller ID (Keine monatliche Gebühr)

Sie können eine bestehende Nummer für **ausgehende Anrufe** verifizieren—keine eingehende Weiterleitung.

- **Verifizierung**: Wir bestätigen das Eigentum über einen kurzen OTP oder Anruf.
- **Pro Minute**: Für die Telefonie, die im Hintergrund verwendet wird, berechnen wir regionsbasierte Raten.
  - **Beispiel**: In den USA könnte Caller ID-Nutzung €0,01/Min betragen.
  - Andere Länder variieren; kontaktieren Sie den Support für Details.

> **Hinweis**: Caller ID allein ermöglicht **keine** eingehende KI; es maskiert nur Ihre ausgehenden Anrufe mit Ihrer verifizierten Nummer.

---

## 4. Internationale Raten & Regionen

Beim Tätigen ausgehender Anrufe:

- **Carrier-Raten**: Einige Länder kosten €0,01/Min, andere €0,45/Min aufgrund lokaler Carrier.
- **Eingehende**: Wenn Sie eine Nummer in einer Hochkosten-Region oder einem speziellen Typ mieten, könnten eingehende Anrufe Zuschläge haben.

Unabhängig davon, ob Sie eine Nummer mieten oder Caller ID/SIP verwenden, hängt der **Carrier-Teil** der Anrufkosten vom Ziel-/Ursprungsland ab und wird typischerweise zu Ihrer Gesamtnutzung hinzugefügt.

---

## 5. Alles zusammen

1. **Nummer mieten**
   - Zahlen Sie eine monatliche Gebühr (ab €3,99), plus Standard-eingehende/ausgehende Carrier-Kosten für Anrufe auf dieser Leitung.
2. **SIP verwenden**
   - Keine monatliche Gebühr von uns, aber eine €0,0004/Min KI-Bridging-Kosten, plus Ihre eigenen PBX-Gebühren.
3. **Caller ID**
   - Keine monatliche Gebühr von uns, aber regionsbasierte Gebühren pro Minute für Telefonie (z.B. €0,01/Min in den USA). Nur ausgehend.
