# Ausgehender Anruf bei Formular-Einreichung

Dieses Tutorial zeigt, wie Sie einen Automatisierungs-Flow einrichten, der einen KI-Anruf auslöst, wenn ein Kunde ein Formular auf Ihrer Website einreicht.

## Was Sie lernen werden

- Einrichten eines Formular-Einreichungs-Triggers
- Konfigurieren der Famulor-Aktion
- Mappen von Formulardaten zu Anruf-Variablen
- Testen und Bereitstellen Ihres Flows
