title: "Runs debuggen"
icon: "bug"
description: "Sicherstellen, dass Ihre Geschäftsautomatisierungen ordnungsgemäß laufen"
---

Sie können jeden Run überwachen, der aus einem aktivierten Flow resultiert:

1. Gehen Sie zum Dashboard, klicken Sie auf **Runs**.
2. Finden Sie den Run, den Sie suchen, und klicken Sie darauf.
3. Sie sehen den Builder im Nur-Lese-Modus, jeder Schritt zeigt ein ✅ oder ein ❌ an, um seinen Ausführungsstatus zu zeigen.
4. Klicken Sie auf einen dieser Schritte, Sie sehen die **Eingabe** und **Ausgabe** im **Run-Details** Panel.
