---

Wenn Sie eine ausgehende Kampagne starten, können Sie wählen, welche Nummer auf den Telefonen der Empfänger erscheint.

## 1. Dedizierte Nummer

- Wenn Sie eine lokale oder gebührenfreie Nummer gemietet haben, können Sie sie als "Telefonnummer" in den Assistenten-Einstellungen auswählen.
- Gut für Markenkonsistenz oder wenn Sie auch eingehende Anrufe auf derselben Leitung annehmen.
- **Keine OTP-Verifizierung erforderlich** für Nummern, die über famulor.de gemietet wurden.

## 2. Verifizierte bestehende Nummer (Nur Anrufer-ID)

- Wenn Sie möchten, dass Ihre persönliche oder Firmen-Telefonleitung erscheint, verifizieren Sie sie mit einem **OTP** oder kurzen Anruf.
- Dies bearbeitet keine eingehenden Anrufe für die KI, aber der Lead sieht diese erkannte Nummer auf der Anrufer-ID.
- **Hinweis**: OTP-Verifizierung ist NUR erforderlich, wenn Sie Ihre eigene bestehende Nummer als Anrufer-ID verwenden.

## 3. SIP-Integration

- Wenn Sie eine PBX oder VOIP-Plattform verwenden, könnten Sie eine benutzerdefinierte Anrufer-ID von Ihrem bestehenden System übergeben.
- Stellen Sie sicher, dass Ihr Trunk oder Anbieter so eingestellt ist, dass eine spezifische ausgehende Anrufer-ID erlaubt ist.

## 4. Länderspezifische Regeln

- Einige Regionen verhindern die Verwendung von Nummern, die Sie nicht besitzen, als Anrufer-ID.
- Stellen Sie immer sicher, dass Sie das Recht haben, diese Nummer zu präsentieren, um Spam oder rechtliche Probleme zu vermeiden.

---

**Tipp:** Testen Sie, indem Sie Ihr eigenes Telefon oder einen freundlichen Kollegen anrufen, um zu bestätigen, dass die richtige Anrufer-ID angezeigt wird, bevor Sie eine große Kampagne starten.
